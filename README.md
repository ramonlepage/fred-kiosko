# FRED | Café & Kiosko

Web oficial de FRED, kiosko de barrio reinventado en L'Eixample, Barcelona.
Café de especialidad · Revistas · Prensa · Cromos · Merch.

## Stack
Sitio estático (un solo `index.html`, sin build). Desplegado en Vercel.

## Desarrollo local
Abrir `index.html` en el navegador, o:
```
npx serve .
```

## Deploy
Cada push a `main` despliega automáticamente en Vercel (una vez conectado el repo).
Deploy manual: `vercel --prod`
